# Weather App 🌦️

## Introduction

The **Weather App** is a Python-based application that allows users to check the current weather conditions of a city.
It retrieves real-time weather data using an API and displays useful information such as temperature and weather conditions.

---

## Features

* Search weather by city name
* Displays temperature and weather description
* Simple and easy-to-use interface

---

## Technologies Used

* Python
* Weather API
* Requests Library

## How to Run the Project

1. Install Python on your system
2. Install required library

```
pip install requests
```

3. Run the program

```
python weather_app.py
```
